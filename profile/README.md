# Mataroa.blog

[Mataroa](https://mataroa.blog/) is a naked blogging platform for minimalists.

Mataroa is:

* open source
* [privacy-conscious](https://mataroa.blog/modus/operandi/)
* [data-interoperability-conscious](https://mataroa.blog/export/)
